### 1. 🎯 타겟 고객 (Customer Segments)

**"누구의 문제를 해결하는가?"**

- **메인 타겟 (Early Adopter):**
    
    - **"증명이 필요한 실행가"**: 학벌/경력보다 '실력'과 '태도'로 평가받고 싶은 취준생, 초기 창업가.
        
    - **"성장 고관여층"**: 자기계발 욕구는 높으나, 구체적인 방법론(Methodology)이 없어 작심삼일로 끝나는 사람들.
        
- **서브 타겟:**
    
    - **인재를 찾는 기업/투자자**: 정량적 스펙(학벌)보다 실제 문제 해결 능력과 성실성을 검증하고 싶은 조직.
        

### 2. 💎 가치 제안 (Value Propositions)

**"왜 우리 서비스를 써야 하는가?"**

- **Scribble to Asset (기록의 자산화)**:
    
    - 부담 없이 던진 '메모(낙서)'를 AI가 구조화된 '회고(자산)'로 변환.
        
    - 기록에 대한 심리적 장벽(Cognitive Load) 최소화.
        
- **Alternative Signal (대체 불가능한 신호 체계)**:
    
    - 일회성 스펙이 아닌, **'꾸준함(Consistency)'**과 **'메타인지(Meta-cognition)'**를 데이터로 시각화하여 증명.
        
    - "이 사람은 1년간 300번의 문제를 정의하고 해결했습니다"라는 인증.
        
- **Community as a Pacer (러닝메이트)**:
    
    - 친목 도모가 아닌, 서로의 성장을 지켜봐 주는 '증인'이자 '속도 메이커' 역할.
        

### 3. 💸 수익 모델 (Revenue Streams)

**"어떻게 돈을 버는가?"**

- **Phase 1 (초기): 유틸리티 & 커뮤니티**
    
    - **멤버십/참가비**: 오프라인 회고 파티, 스쿼드(소모임) 참여 티켓 판매.
        
    - **교육 콘텐츠**: Netty 인증 '나성장 훈련 프로그램' (PDF, VOD, 워크샵).
        
- **Phase 2 (확장): 프리미엄 서비스**
    
    - **전문가 피드백 구독**: AI 분석을 넘어선 휴먼 코치(Admin/Expert)의 1:1 월간 리포트 구독료.
        
- **Phase 3 (완성): HR/매칭 수수료**
    
    - **채용/투자 연계 수수료**: 검증된 인재(높은 등급 유저)를 기업/투자자에 매칭 시 발생하는 수수료.
        

### 4. 📱 서비스 형태 및 채널 (Service & Channels)

**"어떻게 전달하는가?"**

- **접근성 전략 (Low Friction)**:
    
    - **Web App (PWA)**: 앱 설치 불필요. 링크 클릭 즉시 로딩 (React+Vite).
        
    - **Trigger**: 카카오톡 알림톡을 통해 "오늘의 기록" 트리거 발송 → 자동 로그인 링크로 진입.
        
- **UX 컨셉**:
    
    - **입력은 채팅처럼**: 복잡한 양식 지양, 대화하듯 짧게 입력.
        
    - **조회는 대시보드처럼**: 쌓인 데이터는 멋진 그래프와 인사이트로 시각화.

### 5. ⚙️ 핵심 기능 요구사항 (Key Functional Requirements)

#### A. 기록 시스템 (The Engine)

- **[Core] AI 초안 생성기**:
    
    - 사용자가 키워드/단문(낙서) 입력 → AI가 5단계(계기-상황-생각-감정-보완)로 확장 및 문장 정리 제안.
        
- **[Core] 역질문 봇 (Prompting)**:
    
    - 내용이 빈약할 경우 AI가 질문("그때 기분이 어땠나요?")을 던져 구체화 유도.
        
- **[UI] 성장 캘린더**:
    
    - GitHub 잔디 심기 UI 차용. 기록 밀도를 색상으로 표현하여 성취감 부여.
        

#### B. 신호 체계 (The Signal)

- **[Data] 성장도감 (Leveling)**:
    
    - 단순 출석수가 아닌, **'기록의 질(AI 분석)' + '지속성' + '동료 인증'**을 합산한 Netty 독자 등급 산정.
        
- **[View] 역량 포트폴리오**:
    
    - 사용자의 강점 키워드(예: #실행력 #위기관리)를 AI가 추출하여 워드클라우드/차트로 시각화.
        

#### C. 커뮤니티 & 관리 (The Support)

- **[Social] 스쿼드(Squad) & 인증**:
    
    - 비슷한 목표를 가진 유저끼리 그룹핑.
        
    - 서로의 회고에 '봤어요/멋져요/궁금해요' 등으로 가볍게 반응(Witnessing).
        
- **[Admin] CRM 대시보드**:
    
    - 관리자가 전체 회원의 기록 현황 모니터링.
        
    - '이탈 위험군', '성장 정체군' 자동 필터링 및 수동 개입(메시지 발송).
        

#### D. 인프라 & 데이터 (Tech & Data)

- **Frontend**: React, Vite, TailwindCSS (모바일 최적화), PWA 설정.
    
- **Backend**: Node.js, Express, Passport.js (Auth).
    
- **DB**: MongoDB (비정형 텍스트 데이터 저장 및 유연한 스키마 확장에 유리).
    
- **AI**: OpenAI API (GPT-4o-mini 등 비용 효율적 모델 사용).
